College-Directory-Application

  • Tech Stack: Java, J2EE, Spring Boot,Swagger, Postman
  • Github URL: Project Link

I Develop a College Directory Application to facilitate seamless interaction between students, faculty members, and administrators within a single college. The application will allow users to manage and access personal and academic information efficiently.

Designed and implemented user interfaces for different roles: Student, Faculty, and Administrator.

Utilized Spring Boot for backend development and MYSQL for database management

Created login and authentication functionality, validating user credentials and redirecting based on user roles.

Implemented profile management for users, allowing students to view personal and academic details, and faculty to manage class lists and update profiles.

Provided a CRUD interface for administrators to manage student and faculty records.

Designed and implemented API endpoints for user authentication, profile management, searching, and man aging course enrollments.

Implemented a database schema with entities such as User, StudentProfile, FacultyProfile, AdministratorPro file, Course, Enrollment, and Department.

Set up and managed many-to-many relationships between students and courses, and one-to-many rela tionships for users, departments, and faculty.

Designed a secure authentication system for users using role-based access control.

Future improvements include implementing a frontend interface using React and enhancing security with JWT tokens.
